xml2dci
=======

Trivial script to extract DCI numbers from a DCI Reporter xml file.

Usage: xml2dci.py source.xml <destination.csv>

Options:
  -h, --help          show this help message and exit
  -j, --judges        includes judges in the csv
  -s, --scorekeepers  includes scorekeepers in the csv
  -n, --no-players    skips players in the csv
